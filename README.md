# NPD Fuel Monitoring System

AI-powered mobile app for fuel monitoring and management — NPD Ltd Rwanda.

## Problem
Monthly fuel allocations are misused. No tracking exists. This system adds
full accountability: requests, approvals, receipts, odometer logs, and AI anomaly detection.

## Stack
- Backend: NestJS + PostgreSQL + Prisma
- Mobile: Flutter
- AI: FastAPI + Tesseract + Scikit-learn
- Storage: Cloudinary
